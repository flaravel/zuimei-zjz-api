# 证件照制作

当用户需要制作证件照时：

**用户意图**：
- "帮我制作一张证件照"
- "我需要一寸照片"
- "帮我生成蓝底证件照"

**处理流程**：

1. 确认规格：一寸(295x413)/二寸(413x579)/护照(390x567)
2. 确认背景色：白底(#FFFFFF)/蓝底(#438EDB)/红底(#FF0000)
3. 调用 API：`/api/v1/photo/id-photo`

**常见规格**：

| 类型 | 宽 x 高 (px) | 毫米 |
|------|-------------|------|
| 一寸 | 295 x 413 | 25 x 35 |
| 二寸 | 413 x 579 | 35 x 49 |
| 护照 | 390 x 567 | 33 x 48 |

**示例代码**：

```python
result = client.id_photo(
    "photo.jpg",
    width=295,
    height=413,
    background_color="#438EDB",
    beautify=False  # 默认不开启美颜
)
```
