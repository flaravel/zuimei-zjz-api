# 排版照

当用户需要将证件照排版打印时：

**用户意图**：
- "帮我把证件照排成6寸的"
- "我需要打印排版"
- "生成冲印排版"
- "排版成一版多张"

**处理流程**：

1. 确认排版规格：6寸/5寸/A4
2. 确认是否需要裁切线
3. 调用 API：`/api/v1/photo/layout`

**API 参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| image | file | 是 | 证件照图片 |
| layout_type | string | 否 | 排版类型：6inch/5inch/a4，默认 6inch |
| crop_line | boolean | 否 | 是否添加裁切线，默认 false |

**排版规格**：

| 类型 | 尺寸 | 说明 |
|------|------|------|
| 6inch | 6寸（4x6英寸） | 最常用，适合一寸/二寸 |
| 5inch | 5寸（3.5x5英寸） | 小尺寸排版 |
| a4 | A4纸 | 大量排版 |

**排版数量**：

| 照片规格 | 6寸排版数量 |
|----------|------------|
| 一寸 | 8张 |
| 二寸 | 6张 |
| 护照照 | 4-6张 |

**示例代码**：

```python
# 6寸排版（默认）
result = client.layout("id_photo.jpg")

# 6寸排版 + 裁切线
result = client.layout("id_photo.jpg", crop_line=True)

# A4排版
result = client.layout("id_photo.jpg", layout_type="a4")
```

**适用场景**：
- 证件照打印
- 冲印店冲印
- 自助打印

**注意事项**：
- 输入图片需为标准证件照
- 裁切线方便手工裁剪
- 输出为高分辨率图片，适合打印
