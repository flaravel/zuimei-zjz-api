# 智能抠图

当用户需要将人物从背景中分离时：

**用户意图**：
- "帮我把这张照片的背景去掉"
- "我需要透明背景的人像"
- "抠图，把人抠出来"
- "移除背景，只要人物"

**处理流程**：

1. 确认图片中是否有人像
2. 调用 API：`/api/v1/segment/portrait`

**API 参数**：

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| image | file | 是 | 原始图片文件 |

**返回结果**：
- 透明背景的 PNG 图片（Base64 编码或 URL）

**示例代码**：

```python
# 智能抠图
result = client.segment_portrait("photo.jpg")
# 返回透明背景 PNG
```

**适用场景**：
- 证件照前期处理
- 头像制作
- 海报设计
- 电商产品图

**注意事项**：
- 图片需包含清晰的人像
- 支持 JPG、PNG 格式
- 图片大小不超过 10MB
